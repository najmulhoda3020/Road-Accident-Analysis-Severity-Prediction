# Traffic-Accident-Severity-and-Hotspot-Detection
End-to-end ML pipeline on US Accidents dataset to detect road accident hotspots and predict severity. Uses Random Forest, XGBoost &amp; Logistic Regression with SMOTE balancing, K-Means clustering, Folium geospatial maps, Plotly trends &amp; a Streamlit dashboard. Built on Google Colab.
